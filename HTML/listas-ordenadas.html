Listas Ordenadas

Para criar uma lista ordenada, utilizamos o elemento ol. Os itens de uma lista ordenada são definidas utilizando o elemento li. Normalmente, os navegadores definem todos esses elementos como block-level elements.

Exemplo:

Macarrão Instantâneo - K19 Treinamentos

  1. Ferver 600ml de água em uma panela
  2. Retirar o macarrão do pacote
  3. Colocar o macarrão na panela com o fogo baixo
  4. Cozinhar o macarrão por 3 minutos
  5. Temperar a gosto
